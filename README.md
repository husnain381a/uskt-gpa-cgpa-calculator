# USKT GPA & CGPA Calculator

A GPA & CGPA calculator built for students of the **University of Sialkot (USKT)**.

## Features

- **Semester GPA Calculator** – Calculate your GPA for a single semester using the official USKT grading scale
- **Cumulative CGPA Calculator** – Compute your overall CGPA across multiple semesters
- **Performance Dashboard** – Animated results with score category, star rating, circular score display, and motivational quotes
- **USKT Grading Reference** – Full grading policy modal (marks ranges, grade points, formulas)
- **Responsive Design** – Works on mobile and desktop with a dark gradient theme
